COS'È UN BOT TELEGRAM
  • Account automatizzato con cui è possibile
    interagire come con un qualunque contatto
  • Gestito da un programma in esecuzione su
    un computer, come un server web
    • scritto in Python, potenzialmente containerizzabile per es. con Docker
  • Informazioni sul servizio ottenute dai server
    esistenti di TPL FVG
    • grazie agli API attivi; alcune informazioni assenti ottenute dal sito
tplfvgbot • Andrea Esposito
  • Interazione tramite comandi, riconosciuti dal bot,
    cui è associata una risposta statica o dinamica
    • preceduti da uno slash e cliccabili o selezionabili dal menu del bot
  • in questo caso, il comando start mostra un
    messaggio esplicativo del funzionamento del bot
    • risposta statica
  • scrivendo il codice di una fermata, è possibile consultarne
    l'orario in tempo reale
    • altri metodi di ricerca descritti successivamente
tplfvgbot • Andrea Esposito
  • ricerca fermata per nome
    • risultati con linee e direzione, per individuare la fermata corretta
  • codici delle fermate cliccabili
    • per evitare di riscrivere il codice fermata appena trovato
