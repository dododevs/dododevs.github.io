Profile

Laureando in informatica con il pallino dello sviluppo software; background scientifico e discreta esperienza in campo desktop, principalmente con Java, mobile con Java + Android SDK e server con Python e Django, rafforzata e messa in pratica durante il percorso di studio universitario di informatica pura con indirizzo Software and Systems Engineering, spesso in team con altri studenti. Esperienza sul campo acquisita, oltre che attraverso la realizzazione di diversi progetti personali, con lavoro autonomo di consulenza software a singoli o team e con contributi a progetti open source, di cui attivo fautore. Utente Linux, di cui entusiastico testimone.

Computer Science undergraduate with a thing for software development; scientific background and a fair experience with desktop programming, mainly Java, mobile with Java + Android SDK and server-side programming with Python and Django, strengthened and put into practice during the course of study of Computer Science with the Software and Systems Engineering specialisation, often teamed with fellow students. Alongside building a number of personal projects, hands-on experience was acquired with freelance jobs of software consulting for individuals or teams and contributions to open source projects, actively endorsed. Linux user, enthusiast, ambassador.
